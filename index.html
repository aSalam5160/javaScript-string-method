Extracting string part!

There are three methods extracting a part of a string. The slice(), The substring() and The substr(). All method are pretty much similar. All methods has two parameter. The difference is- In slice(), first parametere used for start extracting part and second one used for finish extracting part(Second parametere doesn't included). If the parameter value is negetive the string started extracting from the end og the string. In substring(), if the parameter is negetive the parameter used as ZERO. If you omit the second parameter the method will slice out the rest of the string. In substr(), the 2nd parameter is used for the extracting length.

Converting a string with an array

If you want to work with a string as an array you can convert it an array. The method called split. Arrays splited depends on spaces, comma, pipe. If you ommited the separtor it'll contain the full string as it's zero index.

  1. let x = "a,b,c" ;
    js-code: x.split(",");
    Now x is treat an array with index [0] = a, [1] = b and [2] = c;

  2. let x = "ab cd e" ;
    js-code: x.split(" ");
    Now x is treat an array with index [0] = ab, [1] = cd and [2] = e;

  3. let x = "ami|tumi|she" ;
    js-code: x.split("|");
    Now x is treat an array with index [0] = ami, [1] = tumi and [2] = she;

  4. let x = "a,b,c" ;
    js-code: x.split("");
    Now x is treat an array with index [0] = a,b,c;
